# 高校软件小组管理系统 - 启动指南

## 📋 项目信息

- **项目名称**: 高校软件小组管理系统
- **技术栈**: JSP + Servlet + JDBC + MySQL + Maven
- **项目路径**: `E:\java\Software_group`
- **数据库**: MySQL (software_group)

## 🚀 快速启动方案

### 方案一：使用Maven内嵌Tomcat（推荐）

这是最简单的启动方式，无需单独安装Tomcat。

#### 步骤：

1. **打开CMD命令行**
   - 按 `Win + R`，输入 `cmd`，回车

2. **进入项目目录**
   ```cmd
   cd /d E:\java\Software_group
   ```

3. **编译并运行项目**
   ```cmd
   mvn clean tomcat7:run
   ```

4. **访问系统**
   - 等待控制台显示 `Server startup in XXX ms`
   - 浏览器访问：http://localhost:8080/software-group/
   - 默认管理员账号：
     - 用户名: admin
     - 密码: admin123

#### 常用命令：

```bash
# 编译项目
mvn clean compile

# 打包为war
mvn clean package

# 运行内嵌Tomcat
mvn tomcat7:run

# 后台运行
mvn tomcat7:run -DskipTests
```

---

### 方案二：使用独立Tomcat部署

如果你已经安装了Tomcat，可以使用此方案。

#### 前提条件：

1. 已安装 Apache Tomcat 8.5 或更高版本
2. 已配置好 JAVA_HOME 环境变量

#### 步骤：

1. **编译打包项目**
   ```cmd
   cd /d E:\java\Software_group
   mvn clean package
   ```

2. **复制war包到Tomcat**
   ```cmd
   copy target\software-group.war C:\apache-tomcat-9.0\webapps\
   ```

3. **启动Tomcat**
   ```cmd
   C:\apache-tomcat-9.0\bin\startup.bat
   ```

4. **访问系统**
   - http://localhost:8080/software-group/

## 🔧 常见问题

### 1. 端口被占用

如果8080端口被占用，可以修改端口：

**方案一：Maven内嵌Tomcat**
```cmd
mvn tomcat7:run -Dmaven.tomcat.port=8081
```

**方案二：修改配置文件**
编辑 `src/main/resources/config.properties`：
```properties
server.port=8081
```

### 2. 数据库连接失败

检查以下几点：
- MySQL服务是否启动
- db.properties中的密码是否正确
- 数据库 `software_group` 是否存在

### 3. 编译失败

检查以下几点：
- Java版本（需要1.8+）
- Maven是否安装正确
- 网络连接（下载依赖需要联网）
