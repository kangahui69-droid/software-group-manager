# AI助手知识库 - 系统使用指南

## 一、用户角色说明

| 角色 | 说明 | 标识 |
|------|------|------|
| 游客 (GUEST) | 未登录用户，可浏览公开信息 | 无账号 |
| 成员 (MEMBER) | 已登录的普通成员，可使用成员功能 | 个人中心 |
| 管理员 (ADMIN) | 系统管理员，可使用所有管理功能 | 管理中心 |

---

## 二、功能权限矩阵

### 2.1 游客可使用功能

| 功能 | 操作说明 |
|------|----------|
| **浏览新闻** | 访问首页或通知公告、获奖新闻、活动新闻页面 |
| **提交问题反馈** | 点击导航"问题反馈"，填写标题和内容提交 |
| **提交招新报名** | 访问招新报名页面，填写信息提交申请 |
| **登录/注册** | 使用学号和密码登录，新成员由管理员审核后创建账号 |
| **AI对话** | 可向AI助手咨询基本问题 |

---

### 2.2 成员可使用功能

#### 个人信息管理
| 功能 | 操作说明 |
|------|----------|
| **个人中心** | 点击右上角头像进入，可查看个人信息和快捷入口 |
| **编辑资料** | 在个人中心点击"编辑资料"修改个人信息 |
| **修改密码** | 在个人中心点击"修改密码" |

#### 活动相关
| 功能 | 操作说明 |
|------|----------|
| **查看活动列表** | 点击导航"活动"查看所有活动 |
| **报名参加活动** | 在活动详情页点击"报名"，需在报名时间内 |
| **查看我的活动** | 在个人中心点击"我的活动"查看报名记录 |
| **取消报名** | 在我的活动页面点击"取消报名" |

#### 项目相关
| 功能 | 操作说明 |
|------|----------|
| **创建项目** | 点击"我的项目"中的"创建项目"按钮 |
| **查看我的项目** | 在个人中心点击"我的项目"查看申请记录 |

#### 奖项相关
| 功能 | 操作说明 |
|------|----------|
| **提交奖项申请** | 点击"我的奖项"中的"提交奖项"按钮，上传证书 |
| **查看我的奖项** | 在个人中心点击"我的奖项"查看获奖记录 |

#### 学习相关
| 功能 | 操作说明 |
|------|----------|
| **学习中心签到** | 点击导航"学习中心"，在规定时间（6:00-22:00）签到 |
| **签退** | 在学习中心点击"签退"结束学习，系统18:00自动签退 |
| **查看学习记录** | 在个人中心可查看学习时长统计 |

#### 问题反馈
| 功能 | 操作说明 |
|------|----------|
| **提交问题反馈** | 点击导航"问题反馈"，填写标题和内容 |
| **查看我的反馈** | 在问题反馈页面查看历史反馈记录和处理状态 |

#### AI对话
| 功能 | 操作说明 |
|------|----------|
| **AI助手** | 点击导航"AI助手"进入对话页面，可咨询系统使用问题 |

---

### 2.3 管理员可使用功能

#### 管理后台入口
- 顶部导航点击"管理中心"或右上角头像下拉菜单
- 或点击左侧菜单图标展开侧边栏快捷入口

#### 新闻管理
| 功能 | 操作说明 |
|------|----------|
| **发布新闻** | 点击"新闻管理" → "发布新闻"，选择类型（通知/获奖/活动）填写内容 |
| **编辑新闻** | 在新闻列表点击"编辑"修改 |
| **删除新闻** | 在新闻列表点击"删除"（已关联活动的新闻无法删除） |

#### 活动管理
| 功能 | 操作说明 |
|------|----------|
| **创建活动** | 点击"活动管理" → "创建活动"，填写时间地点和报名时间 |
| **管理报名** | 点击活动列表的"报名管理"查看/审核报名 |
| **删除活动** | 点击"删除"移除活动（进行中/已结束的活动不可删除） |

#### 项目管理
| 功能 | 操作说明 |
|------|----------|
| **审核项目** | 在项目管理页面查看待审核项目，点击"批准"或直接编辑 |
| **查看项目详情** | 点击项目列表的"编辑"查看详细信息 |
| **删除项目** | 点击"删除"移除项目 |

#### 奖项管理
| 功能 | 操作说明 |
|------|----------|
| **审核奖项申请** | 在奖项管理页面查看待审核申请，点击"通过"或"拒绝" |
| **管理奖项** | 可查看、编辑已通过/拒绝的奖项记录 |

#### 成员管理
| 功能 | 操作说明 |
|------|----------|
| **查看成员列表** | 点击"成员管理"查看所有成员 |
| **重置成员密码** | 点击成员"重置密码"，默认密码为123456 |

#### 招新管理
| 功能 | 操作说明 |
|------|----------|
| **处理报名申请** | 点击"招新管理"查看待处理申请 |
| **同意报名** | 点击"同意"为申请者创建账号（学号为用户名，默认密码123456） |
| **拒绝报名** | 点击"拒绝"驳回申请 |

#### 学习管理
| 功能 | 操作说明 |
|------|----------|
| **查看学习记录** | 点击"学习管理"查看所有成员的学习时段记录 |
| **筛选查询** | 按日期范围和成员筛选学习记录 |
| **统计数据** | 查看总学习次数、总时长、平均时长等统计 |

#### 问题管理
| 功能 | 操作说明 |
|------|----------|
| **查看问题反馈** | 点击"问题管理"以三个标签页查看（属实/不属实/待确认） |
| **处理问题** | 点击"查看"可更新分类、状态和备注 |
| **删除问题** | 点击"删除"移除问题记录 |

#### 系统日志
| 功能 | 操作说明 |
|------|----------|
| **查看操作日志** | 点击"日志"查看用户操作记录 |

#### AI助手
| 功能 | 操作说明 |
|------|----------|
| **AI对话** | 与普通成员相同，可咨询系统问题 |
| **AI统计** | 点击查看AI助手使用统计和常见问题 |

## 三、快捷操作指南

### 3.1 常用操作路径

| 操作 | 路径 |
|------|------|
| 登录系统 | 首页点击"登录"，输入学号和密码 |
| 报名活动 | 活动列表 → 选择活动 → 点击"报名" → 确认 |
| 提交奖项 | 我的奖项 → 提交奖项 → 填写信息 → 上传证书 |
| 创建项目 | 我的项目 → 创建项目 → 填写信息 → 提交 |
| 学习签到 | 学习中心 → 点击"开始学习"（6:00-22:00） |
| 提交问题 | 问题反馈 → 填写标题和内容 → 提交 |

### 3.2 AI助手可回答的问题类型

#### 游客可咨询
- 如何注册/登录
- 如何报名参加活动
- 有哪些新闻和活动
- 如何提交问题反馈
- 招新报名相关问题

#### 成员可咨询
- 如何修改个人资料
- 如何报名参加活动
- 如何提交奖项申请
- 如何创建项目
- 如何编辑个人简历
- 学习签到/签退规则
- 问题反馈相关问题

#### 管理员可咨询
- 如何发布新闻
- 如何审核奖项申请
- 如何创建活动
- 如何处理招新报名
- 如何重置成员密码
- 如何管理项目申请
- 问题反馈处理流程
- 学习管理相关问题

## 四、状态说明

### 4.1 活动报名状态
| 状态 | 说明 |
|------|------|
| 报名中 | 在报名时间内，可以报名 |
| 已结束 | 报名时间已截止 |
| 未开始 | 报名尚未开始 |

### 4.2 项目状态
| 状态 | 说明 |
|------|------|
| 待审核 | 等待管理员审批 |
| 已通过 | 审批通过 |
| 已拒绝 | 审批被拒绝 |

### 4.3 奖项状态
| 状态 | 说明 |
|------|------|
| 待审核 | 等待管理员审批 |
| 已通过 | 审批通过，荣誉已录入 |
| 已拒绝 | 审批被拒绝 |

### 4.4 问题反馈状态
| 分类 | 说明 |
|------|------|
| 属实 | 问题确实存在，需要处理 |
| 不属实 | 问题经验证不属实 |
| 待确认 | 待管理员验证 |

| 状态 | 说明 |
|------|------|
| 待处理 | 属实问题待处理 |
| 正在解决 | 问题正在处理中 |
| 已解决 | 问题已处理完成 |

## 五、默认信息

| 项目 | 默认值 |
|------|--------|
| 新成员密码 | 123456 |
| 学习时间 | 6:00 - 22:00 |
| 强制签退时间 | 18:00 |
| 自动结束学习 | 22:00 |
| 早到签到时间 | 6:00 - 18:00 |
| 正常签到时间 | 18:00 - 19:00 |
| 迟到签到时间 | 19:00 之后 |

---

## 六、注意事项

1. **密码安全**: 首次登录后请立即修改密码
2. **活动时间**: 活动报名必须在报名时间内进行
3. **学习规则**: 18:00后签到记为迟到，22:00系统自动结束学习
4. **审核流程**: 项目、奖项等需要管理员审核后才能生效
5. **问题反馈**: 请详细描述问题，以便管理员及时处理
