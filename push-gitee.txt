【推荐使用】个人访问令牌
========================================

为了安全，建议使用个人访问令牌代替密码：

1. 登录Gitee (https://gitee.com)
2. 点击右上角头像 → 设置
3. 左侧菜单 → 私人令牌
4. 点击「+生成新令牌」
5. 输入令牌描述（如：SoftwareGroup推送）
6. 勾选权限：projects（查看、编辑项目）
7. 点击「提交」，复制生成的令牌
8. 在上面的Password处粘贴该令牌

========================================
【常见问题】问题排查
========================================

Q1: 提示 "fatal: unable to access..."
A: 检查网络连接，确认能访问 https://gitee.com

Q2: 提示 "Authentication failed"
A: 用户名或密码/令牌错误，请重新输入

Q3: 提示 "failed to push some refs"
A: 远程仓库已有代码，执行：
   git pull origin develop --allow-unrelated-histories
   然后再推送

Q4: 提示 "could not resolve host"
A: DNS解析失败，尝试：
   git push https://IP地址替换/hsuniversity/softwaregroup.git master:develop
