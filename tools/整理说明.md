## 使用方法

### 1. 密码修复工具

当遇到登录密码问题时使用：

```bash
# 进入项目目录
cd F:\Software_group

# 编译并运行修复工具
mvn exec:java -Dexec.mainClass="util.DirectPasswordFix"
```

### 2. 诊断工具

当需要诊断系统问题时使用：

```bash
# 运行诊断工具
mvn exec:java -Dexec.mainClass="util.LoginDiagnosticTool"
```

## 注意事项

1. **备份数据**：运行任何工具前请备份数据库
2. **测试环境**：先在测试环境验证工具效果
3. **查看日志**：工具运行时会输出详细日志
4. **权限要求**：某些工具需要数据库写权限
